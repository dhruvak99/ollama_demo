# ollama_demo
AI WORKSHOP
Setup Anaconda (First)

Download Anaconda:
https://www.anaconda.com

Create Environment
conda create -n ollama python=3.11 -y

Activate Environment
conda activate ollama

⸻

Setup Ollama

Download Ollama:
https://ollama.com

Verify Installation
ollama –version

Run a Model
ollama run qwen3:4b

⸻

Run Python Scripts

Install dependency
pip install requests

Run Script
python ollama_local_chat_api.py

⸻

Setup OpenWebUI

Project:
https://github.com/open-webui/open-webui

Install
pip install open-webui

Run
open-webui serve

Open in Browser
http://localhost:8080

⸻

RAM vs Model Size

Approximate RAM required for quantized models:

1B parameters ≈ 0.6 – 0.8 GB RAM

Examples

3B → 2 – 3 GB
4B → 3 – 4 GB
7B → 5 – 8 GB
13B → 10 – 16 GB
34B → 25 – 40 GB
70B → 50 – 80 GB
